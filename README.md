# UAV Route Planner v3

Полнофункциональное приложение для планирования маршрутов беспилотных летательных аппаратов (БПЛА) с оптимизацией по различным алгоритмам.

## Функциональность

- **Интерактивная карта** с рисованием контура поля
- **Добавление препятствий**: деревья, ЛЭП, запретные зоны, здания
- **Расчёт полос съёмки** с учётом параметров камеры и перекрытия
- **Оптимизация маршрута** четырьмя алгоритмами:
  - DP (Dynamic Programming / TSP с 2-opt)
  - VRP (Vehicle Routing Problem)
  - GA (Genetic Algorithm)
  - ACO (Ant Colony Optimization)
- **Визуализация**:
  - NDVI подложка (синтетическая)
  - DEM горизонтали высот
  - Буферные зоны препятствий
  - Маршруты с учётом обхода препятствий
- **Сравнение алгоритмов** на одном наборе данных
- **Экспорт путевых точек** в CSV
- **Синтетические данные** NDVI и DEM для демонстрации

## Использование

### 1. Добавление контура поля
- Нажмите **"+ Точка поля"**
- Кликайте по карте для добавления вершин контура
- Минимум 3 точки
- Можно вводить координаты вручную в левой панели

### 2. Добавление препятствий
- Выберите тип препятствия из списка
- Нажмите **"+ Препятствие"**
- Кликайте на карту:
  - **Дерево**: 1 клик
  - **ЛЭП**: 2 клика
  - **Запретная зона / Здание**: несколько кликов, двойной клик для завершения

### 3. Настройка параметров БПЛА
- **Высота (м)**: высота полёта
- **Скорость (м/с)**: скорость движения
- **Угол обзора (°)**: угол обзора камеры
- **Батарея (мин)**: время автономного полёта
- **Перекрытие вдоль (%)**:продольное перекрытие
- **Перекрытие поперёк (%)**: боковое перекрытие

### 4. Выбор алгоритма
- **DP**: быстрый, хороший результат (рекомендуется)
- **VRP**: несколько независимых маршрутов
- **GA**: генетический алгоритм
- **ACO**: муравьиный алгоритм

### 5. Расчёт маршрута
- Нажмите **"▶ Рассчитать"**
- Дождитесь завершения расчёта
- Просмотрите результаты в левой панели

### 6. Сравнение алгоритмов
- Нажмите **"⚖ Сравнить все"**
- Приложение запустит все 4 алгоритма
- Результаты появятся на вкладке "Сравнение"

### 7. Просмотр результатов
- **Вкладка "Карта маршрута"**: визуальное представление
- **Вкладка "NDVI"**: синтетический индекс растительности
- **Вкладка "DEM"**: цифровая модель рельефа
- **Вкладка "Путевые точки"**: таблица координат
- **Вкладка "Сравнение"**: результаты сравнения алгоритмов

### 8. Экспорт данных
- Нажмите **"💾 Экспорт CSV"** на вкладке "Путевые точки"
- CSV файл содержит координаты всех путевых точек

## Алгоритмы оптимизации

### DP (Dynamic Programming / TSP)
- **Описание**: классический алгоритм коммивояжёра с оптимизацией 2-opt
- **Преимущества**: быстрый, стабильный результат
- **Недостатки**: может застрять в локальном оптимуме
- **Время**: <1 сек для 100 полос

### VRP (Vehicle Routing Problem)
- **Описание**: разбиение маршрута на несколько независимых путей
- **Преимущества**: поддержка множественных вылетов
- **Недостатки**: может быть менее оптимальным
- **Параметры**: количество маршрутов

### GA (Genetic Algorithm)
- **Описание**: эволюционный алгоритм с кроссовером и мутацией
- **Преимущества**: может найти лучшее решение
- **Недостатки**: медленнее, требует настройки параметров
- **Параметры**: размер популяции, количество поколений

### ACO (Ant Colony Optimization)
- **Описание**: имитация поведения муравьёв
- **Преимущества**: хорошее качество решения
- **Недостатки**: медленнее, требует больше вычислений
- **Параметры**: количество муравьёв, количество итераций

## Параметры препятствий

- **Радиус буфера**: расстояние обхода препятствия
- **Штраф буфера**: увеличение стоимости маршрута при пересечении буфера (5x)
- **Минимальная ширина сегмента**: 1 метр

## Горячие клавиши

- **Enter**: подтвердить ввод координат
- **Ctrl+Z**: отменить (не реализовано)
- **Ctrl+S**: сохранить (не реализовано)
- **F1**: справка (не реализовано)

## Известные ограничения

- Максимум 1000 полос съёмки
- Максимум 50 препятствий
- NDVI и DEM имеют разрешение 180x180 пикселей (синтетические)
- Не поддерживается импорт реальных данных DEM
